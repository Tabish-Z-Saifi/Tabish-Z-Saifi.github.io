How syncing works

  1. Export your data on the device you’ve been using — a .json file is saved/shared.
  2. Send it to yourself via AirDrop, WhatsApp, or email.
  3. Open this app on the other device and tap Import Data, pick the file.
  4. Choose Merge (combine) or Replace (overwrite) when asked.

Tip: there’s no automatic cloud sync — always export the latest before switching devices.
