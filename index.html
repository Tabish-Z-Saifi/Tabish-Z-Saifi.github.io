One-time setup needed

This app needs a (free) Firebase project to sync your guest list across devices.

  1. Go to console.firebase.google.com → Add project
  2. Authentication → enable Email/Password and Google
  3. Firestore Database → Create database
  4. In Firestore → Rules, paste the contents of firestore.rules → Publish
  5. Project settings → Your apps → Web → copy the config into js/config.js
  6. Authentication → Settings → Authorized domains → add your GitHub Pages domain

Full steps are in README.md. Reload this page when done.
